# Code of Conduct

## Our Standards

SWIM-RS is committed to providing a welcoming and respectful environment for all contributors, regardless of background or experience level.

We expect all participants to:

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Accept constructive criticism gracefully
- Focus on what is best for the community and the project
- Show empathy towards other community members

## Scope

This Code of Conduct applies to all project spaces, including the GitHub repository, issue tracker, pull requests, and any other communication channels associated with SWIM-RS.

## Enforcement

Instances of unacceptable behavior may be reported by contacting the project maintainer. All reports will be reviewed and investigated, and will result in a response deemed necessary and appropriate to the circumstances.
